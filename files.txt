Data file-i in njihova uporaba


data/heatmap_sorted.csv
Stolpci: ggo, odsek_id, leto_mesec, target

Napove/izmeri aktivnost poseka po odseku in mesecu (vrednost target)
Server izračuna kvantilne meje in vsak odsek razvrsti v bucket 0–4
Frontend bere /api/heatmap?month=... in s funkcijo buildHeatmapExpression() obarva vsak odsek na mapi z eno od 5 barv heatmape
Barve so definirane v styles.css:3-7:
bucket 0 → #22c55e (zelena, ni aktivnosti)
bucket 1 → #84cc16
bucket 2 → #facc15 (rumena)
bucket 3 → #f97316 (oranžna)
bucket 4 → #ef4444 (rdeča, visoka aktivnost)
Slider za mesec (/api/heatmap/meta) prikazuje kdaj so historični vs. napovedani podatki (meja: FORECAST_START_MONTH = '2025-01')


data/posek_processed.csv
Stolpci: ggo, odsek, manual, kubikov, vrsec, posekano, ko, parcela, leto, mesec

Vsebuje dejanski posek v kubičnih metrih po odseku, mesecu in vzroku (vrsec)
Prikaže se v levem panelu ko izbereš odsek + mesec: skupna količina m³ in po kategorijah (Sanitarni/Redni/Ostalo)
Relevantna API pot: /api/posek?odsek=...&month=...
Prikazuje se samo če total_kubikov > 0


data/odseki_nazivi.csv
Stolpci: ggo_naziv, odsek, povrsina, gge_naziv, ke_naziv, revir_naziv, katgozd_naziv, ohranjen_naziv, relief_naziv, lega_naziv, pozar_naziv, intgosp_naziv, krajime, grt1_naziv

Opisni atributi vsakega odseka (ime, površina, GGE, revir, kategorija gozda, ...)
Prikazuje se v details tabeli v levem panelu ko klikneš/poiščeš odsek
Poganja tudi autocomplete/suggest iskanje po odsek ID
Napaja /api/ggo (seznam GGO območij) in /api/odseki/suggest


data/odseki_vector_map.mbtiles
Vektorske tile-e za odseke — geometrije poligonov na mapi
Servira se na /tiles/{z}/{x}/{y}
Uporablja se za: risanje odsekov, klikanje, heatmap barvanje (fill-color), highlight izbranega odseka
Ob zagonu server iz teh tiles zgradi bbox cache za zoom na odsek


data/ggo_vector_map.mbtiles
Meje GGO območij (gozdnogospodarska območja)
Servira se na /ggo-tiles/{z}/{x}/{y}
Prikazan kot zelena (#4ade80) obroba na mapi — layer ggo-outline
Toggle z checkboxom v legendi


data/slovenija_vector_map.mbtiles
Meja Slovenije
Servira se na /slo-tiles/{z}/{x}/{y}
Prikazan kot svetlo modra (#60cdee) obroba — layer slovenija-outline
Toggle z checkboxom v legendi


data/odseki_processed.csv
Vsebuje surove geometrije + vse numerične atribute odsekov (tarife, lzskdv vrednosti, ...)
Ni direktno v uporabi v spletnem serverju — verjetno vhodni file za generiranje .mbtiles in odseki_nazivi.csv (preprocessing)


data/odseki_vector_map.bbox_cache.json
Cache bounding boxov za vse odseke (generiran iz .mbtiles)
Pospeši zoom na odsek brez ponovnega dekodiranja tiles
